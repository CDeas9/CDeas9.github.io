TravelCast


Takes user's starting location and end location. Then, it gives approximently where the user will be at each hour of their trip and what the weather will be like there. This was created with python and html at a hackathon. It was a group of about 10 people and I worked on the backend using GoogleMaps API. Final WebApp can be found at travelcast.me.

Cyclone


Calculates what level of cyclone, hurricane, and typhoon a storm is based on the user's inputs. This was an indivdual project for class created in Pycharm.

Mortgage


Calculates how many total payments and how much each payment will be for a mortgage based on the user's inputs. This was an indivdual project for class created in Pycharm.

Disney Game


Our final project for programming one was to create our own zuul game. The point of the game was to go through different rooms, collect objects, and win the game. In my game, you are a kid that lost their family in Disney's Magic Kingdom. The characters help you find them if you help the characters. Once you get to the entrance and leave with your whole family, you have won the game. Created in BlueJ alone, but cleaned up in IntelliJ.

Disney Ticket Prices


Written in R using RStudio. I used different models to predict the ticket prices of Disney when new expansions open at the park. The three main variables I found to be predictors are year/inflation, if there was a fan base of the theme before opening, and how many rides were in the expansion.
